# TransferTo

A client for TransferTo API communication

## Installation

Add this line to your application's Gemfile:

    gem 'transfer_to'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install transfer_to

## Usage

@transfer_to_client = TransferToApi::Client.new("account", "password")
